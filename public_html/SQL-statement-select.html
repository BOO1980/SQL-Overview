SQL Select Statement

  • The SQL select statement is used to fetch the data from a database table which returns this data in the form of a result table
  • You can use the select statement on one or more tables
  • To use select to retrieve table data you must, at a minimum specify two pieces of information
    • What you Select
    • And from where you want to select it

Examples

The following example uses the select to retrieve a single column called prod_name from the Products table

select prod_name

from Products;

Retrieving Multiple Columns

  • To retrieve multiple columns from a table, the same select statement is used
  • Multiple columns names must be specified after the select keyword
  • Each column name must be separated by a comma
  • Do not specify a comma after the last column name as it will generate an error

Basic Syntax

Selecting certain columns within at table

select column1, column2, columnN

from Table_name;

Examples

The following code is an example, which would fetch the id, name and salary fields of the customer table.

Retrieving All Columns

  • select statements can also request all columns without having to list them individually
  • Asterisks is a wildcard character that allows you to select all the column names of a table
  • You are better of not using the * wildcard unless you really do need every column in the table as unnecessary columns can slow down the performance of retrieval of data and/or the application
  • The advantage of using wildcards is that it is possible to retrieve columns whose names are unknown

Basic Syntax

Selecting all columns within at table

The wildcard character is specified right after the select statement

select * From
