SQL Create Table

  • The basic syntax to create a table is CREATE TABLE
  • Create table is the keyword telling the database system what you want to do
  • Creating a basic table naming the table and defining its columns and each columns data type
  • A combination of the CREATE TABLE statement and the select statement.
  • A copy of an existing table can be created using a combination of the CREATE TABLE statement and the SELECT statement
  • When you will create a new table using the existing table, the new table would be populated using the existing values in the old table
