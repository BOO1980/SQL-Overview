RDBMS Databases

  • There are many popular RDBMS available to work with.
    • MySQL
      • MySQL is an open source SQL database
      • MySQL is supporting many different platforms including Microsoft Windows, Linux, UNIX and MscOS
      • MySQL comes with a very fast, multi-threaded, multi user and robust SQL database server
    • MS SQL Server
      • MS SQ Server is a relational database management system developed by Microsoft inc
      • The primary query languages are
        • T-SQL
        • ANSI SQL
    • ORACLE
      • A very large multi-user database management system developed by the Oracle Corporation
      • Oracle works to efficiently manage its resources, a database of information among the multiple clients requesting and sending data in the network
    • MS Access
      • This is one of the most popular Microsoft products.
      • Microsoft Access is an entry-level database management software.
      • MS Access database is not only inexpensive but also a powerful database for small-scale projects.
      • MS Access uses the Jet database engine, which utilises a specific SQL language dialect (sometimes referred to as Jet SQL).
      • MS Access comes with the professional edition of MS Office package. MS Access has easy-to-use intuitive graphical interface.
