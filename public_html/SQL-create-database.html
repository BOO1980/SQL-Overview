Create Database

  • The basic syntax of this CREATE DATABASE statement is as follows: CREATE DATABSE DatabaseName
  • Make sure you have the admin privilege before creating any database
  • Once a database is created, you can check it in the list of databases as follows: SHOW DATABASES

Create Table Using Tables

  • A copy of an existing table can be created using a combination of the create table statement and the select statement
  • The new table has the same column definitions. All columns or specific columns can be selected
  • When you will create a new table using the existing table, the new table would be populated using the existing values in the old table
